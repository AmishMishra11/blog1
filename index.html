Best sites that can help you create amazing webpages

7 August, 2021

Here are some of the websites that I use to help me create web pages, so I thought I should share them so that you can use them too and create web pages easily.

List of websites

  1. unDraw
  2. tailwindcss
  3. Unsplash
  4. Google fonts
  5. codepen

unDraw

unDraw is a free-to-use website where you can use to get free images and SVGs that are copyright-free that you can use in your projects. There are a bunch of illustrations to choose from and you can also search for any particular topic and look for that particular SVG. There is also an option to customize colors on your SVG so you can make it look according to your color theme on your website.

visit website

tailwindcss

tailwindcss provides a variety of different colors to choose from and makes that color your theme for your website. It also provides different shades of the same color so you can choose your favorite color. Not only that but you can also create different new colors by combining the base colors that tailwindcss provides. If you are not sure how to do that It also tells you the syntax to use to create a new custom color of your choice. It has many other features that you can try out so visit there and find the perfect theme for your website.

visit website

Unsplash

Unsplash is a website that provides free high-resolution photos that are copyright-free so you can use them on your projects. You can search for a particular topic and look for images related to that topic and unsplash will provide the best images for you. Then you can download that image for free and use that in creating your website.

visit website

Google fonts

Google fonts provide us different fonts that we can use for our typography in our websites. It has all the popular fonts available so you can select one that you like the most and use it for yourself. It also provides icons that you can use on your websites. These are all opensource so you are free to use them without worrying about anything.

visit website

Codepen

CodePen is a social development environment for front-end designers and developers. You can use it to write HTML, CSS, and JS and test how it'll look. It is free to use online code editor that is easy to use, so go ahead and design your first project for free.

visit website

Final thoughts

Using these websites made it so easy for me to design my projects, so I'm sure that they would help you too. Everything is available online nowadays that can help you with anything so go ahead and use the resources that are provided and create amazing projects.
